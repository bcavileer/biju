# Biju

[WIP] Biju is an easy way to mount a GSM modem to send, to receive and to delete messages through a ruby interface.

## Installation

Add this line to your application's Gemfile:

    gem 'biju'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install biju
